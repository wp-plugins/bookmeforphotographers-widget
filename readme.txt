Photographer Appointment Booking System 

== Description ==
Add the Booking Button to your website will increase engagement and create a lasting connection with your audience, also bookmeforphotogaphers.com offers 1 month free trial account.

<strong>Using as a Widget</strong>
<p>When "Bookmeforphotographers Widgets" is activated a widget is also created with title 'Bookmeforphotographers Widgets' you can use that widget to place a book now button on your blog.</p>

<strong>Shortcode Syntax</strong>
<code>[BKFP url='booking url']</code>

 
== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
