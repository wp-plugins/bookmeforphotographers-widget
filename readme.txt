=== Bookmeforphotographers Widgets ===
Contributors: Tiago G
Version: 1.0.0
Create Booking buttons either by using shortcode or use WordPress widgets to place follow button on your blog sidebars.

== Description ==
Add the Booking Button to your website to increase engagement and create a lasting connection with your audience.

<strong>Using as a Widget</strong>
<p>When "Bookmeforphotographers Widgets" is activated a widget is also created with title 'Bookmeforphotographers Widgets' you can use that widget to place follow button on blog sidebars.</p>

<strong>Shortcode Syntax</strong>
<code>[BKFP url='booking url']</code>


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==
**ShortCode Syntax and guide**
Shortcode Syntax
[BKFP url='booking url']
Attributes
{url} - Your booking url.
